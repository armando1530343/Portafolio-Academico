Para el desarrollo de la practia fue necesario el uso de apuntadores en el lenguaje de programacion de c++, ya que sera implementada una aplicacion de Hash Table y esta usa una serie de listas para cada posicion del arreglo. Primeramente se creo la estructura de cada nodo perteneciente al hash table el cual contiene el valor, las posiciones X y Y, un apuntador al nodo siguiente y uno al anterior. Despues fue implementada la funcion hash el cual grafica los nodos con la ayuda de la funcion de GL_POINTS de opengl. Hay un menu con diferentes acciones las cuales crea una nueva tabla, busca un valor en la posicion, inserta un valor en la posicion y remuev un valor de la posicion.
Para crear una tabla se pide un valor por consola entre el rango de 0 a 99, y al momento se crea un arreglo de nodos(nodos H) con la longitud del valor que se ingreso.
Para Buscar un valor se hace un ciclo que recorre todos los nodos cabeza, el cual en cada uno se recorre los valores que fueron insertados y si un valor coicide con la busqueda se pinta de amarillo.
Para insertar un valor primero se verifica en el hash table que no exista para poder insertarlo.
Y para eliminar un valor se hace el mismo proceso de busqueda pero esta ves se quitara al valor y si hay mas valores en ese nodo se recorren los valores delante del numero eliminado.
